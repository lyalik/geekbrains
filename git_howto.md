# Подсказка по GIT:

Создание репозитория
```sh
git init
```
Добавление файла или добавление изминений
```sh
git add
```
Внесение изменений в проект и репу
```sh
git commit -m "Message"
```
Просмотр логов
```sh
git log
```

```sh
git log --oneline
```
Проверка где были изменения
```sh
git checkout
```
Просмсотр изменения файлов.
```sh
git diff
```
